Case Study

AI Trip Purpose Prediction

November 2025 - December 2025

An AI project focused on inferring hidden travel intent from sequential trip behavior using probabilistic sequence models and context-aware feature engineering.

Problem

Predicting intent when labels are not available at runtime

The core challenge in mobility intelligence is that purpose labels (work, home, shopping, social, etc.) are usually available only in curated datasets, not in real-time systems.

This project asks whether hidden intent can be inferred from what is observable: trip mode, ordering, and time context across a day.

Instead of treating each trip independently, we model intent as a sequence problem where adjacent trips provide useful context for decoding the next latent state.

AI Approach

Sequential probabilistic modeling with richer observation signals

We trained multiple sequence baselines including first-order HMM, 3-gram HMM, and edge-emitting HMM, then used Viterbi decoding to infer purpose trajectories on held-out days.

The key modeling decision was feature enrichment at the observation layer: trip mode was combined with time-of-day and hour-of-day buckets to improve state separability.

This converts a weak signal into a more informative emission space, allowing the models to better distinguish transitions that look similar in mode alone but differ by temporal context.

Pipeline steps: GPS trajectories from Geolife were cleaned, grouped by trip, sorted into daily sequences, and filtered for sequence quality before probability estimation.

Initial, transition, and emission distributions were estimated from counts for transparent, interpretable model behavior.

For benchmarking, we compared against non-sequential baselines (home-only, frequency, time-of-day heuristic) and a random forest classifier.

This comparison isolated the value of sequence structure and showed when temporal dependence contributes beyond static classification.

Results

What improved model performance most

  • Temporal context features (especially hour-of-day) materially improved decoding quality.
  • Sequence-aware models outperformed simple heuristics on latent-purpose inference tasks.
  • Higher-order and edge-emitting variants helped test richer transition assumptions.
  • Interpretable probabilistic parameters made error analysis and debugging straightforward.
